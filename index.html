Guia para Criação de Sites

HTML 5

O HTML 5 é uma linguagem de marcação focada no conteúdo do seu site, priorizando a semântica, ou seja, trabalhando com tags que possuam um significado.

CSS 3

As CSS são, como o próprio nome ja diz, folhas de estilo em cascata. Essas folhas são responsáveis pelo design do seu site, alterando tamanhos, formas e cores com o objetivo de deixar sua página mais bonita.

JavaScript

O JavaScript é uma linguagem de programação que no âmbito do desenvolvimento web, é responsável por criar a interatividade do seu site, com animações e alertas por exemplo.
